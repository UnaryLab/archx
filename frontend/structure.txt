Flow
- Add designs
- Create dependencies
- Expand iterations

Add designs

parameters:
    normal: not affected by iteration
    iteration: affected by iteration
    depth_iteration: structure where it is effected by internal iteration (query)
    dependencies: defines interation interaction (none, dependency, antidependency, conditional dependency)

helpers
    parent class:
        - add
        - remove
    child class (implicit by changing parameters)
        - update

typechecking at module for different options

Add dependencies

allow dependencies at 3 levels
    - across designs
    - within designs (workload <-> architecture)
    - within designs (workload <-> workload) || (architecture <-> architecture)

    handle dependency assertions and typechecking specific to dependencies

After dependencies built, now expand and iterate

no dependency: iterates on both dims
dependency: iterates together
antidependency: Do not iterate together
conditional dependency: Condition decides which 3 it is.
